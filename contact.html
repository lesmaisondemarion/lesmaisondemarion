Configurer l’envoi (1 minute)

  1. Va sur formspree.io, crée un compte gratuit et ajoute un formulaire.
  2. Copie l’URL de ton formulaire (ça ressemble à https://formspree.io/f/xxxxx).
  3. Dans ton fichier contact.html, remplace l’attribut action par cette URL (à la place de https://formspree.io/f/REMPLACEZ_VOTRE_ID_ICI).
  4. Commit sur GitHub ; le formulaire sera opérationnel.
